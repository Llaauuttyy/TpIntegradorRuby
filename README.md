# TpIntegradorRuby
## Dependencias
Gema **Mechanize** (Versión 2.7.5)

    sudo gem install mechanize -v 2.7.5
Gema **Nokogiri**

    sudo gem install nokogiri
 También se necesita [**postgresql**](https://www.postgresql.org/).
