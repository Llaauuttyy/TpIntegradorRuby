# TpIntegradorRuby
## Dependencias
### - [**Postgresql**](https://www.postgresql.org/)

### - Gems

Gem **Mechanize** (Versión 2.7.5)

    sudo gem install mechanize -v 2.7.5
Gem **Nokogiri**

    sudo gem install nokogiri

Gem **Pg**

    sudo gem install pg

Gem **Activerecord**

    sudo gem install activerecord
